Persetujuan Data

Ai Pro menggunakan penyimpanan lokal untuk menyimpan riwayat obrolan dan preferensi Anda agar tetap tersedia saat Anda kembali.

Tentang Penyimpanan Kami

Kami sangat menjaga privasi Anda. Berikut adalah cara kami mengelola data Anda:


• Web Storage: Kami menggunakan LocalStorage, bukan cookie pelacak. Data tetap berada di browser Anda dan tidak pernah dikirim ke server kami.

• Riwayat Chat: Pesan disimpan secara lokal agar Anda bisa melanjutkan percakapan nanti.

• Privasi: Karena data bersifat lokal, menghapus cache browser atau menekan "Clear All" di sidebar akan menghapus seluruh data secara permanen.
